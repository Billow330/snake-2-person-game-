# 🐍 双人贪吃蛇游戏

一个可以在本地运行的双人贪吃蛇游戏，支持两条蛇同时游戏。

## 🎮 游戏特性

- **双人模式**：两条蛇同时游戏
- **独立控制**：
  - 蛇1（绿色）：方向键 ↑↓←→ 控制
  - 蛇2（蓝色）：WASD 键控制
- **分数系统**：独立计分，显示最高分
- **现代化UI**：美观的界面设计

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

1. 双击 `启动游戏.bat` 文件
2. 游戏会自动在浏览器中打开
3. 如果浏览器没有自动打开，手动访问：`http://localhost:8000`

### 方法二：手动启动服务器

1. 打开命令行（CMD 或 PowerShell）
2. 进入游戏目录
3. 运行命令：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问：`http://localhost:8000`

### 方法三：直接打开HTML文件

直接双击 `index.html` 文件（某些浏览器可能有安全限制）

## 🎯 游戏规则

- 吃到红色食物可以增长并获得10分
- 撞到墙壁、自己或对方都会失败
- 失败时，另一条蛇获胜
- 显示两条蛇的独立分数和最高分

## 📋 系统要求

- Python 3.x（用于启动本地服务器）
- 现代浏览器（Chrome、Firefox、Edge等）

## 🛠️ 文件说明

- `index.html` - 游戏主页面
- `style.css` - 样式文件
- `game.js` - 游戏逻辑
- `启动游戏.bat` - Windows启动脚本
- `README.md` - 说明文档

## 💡 提示

- 游戏数据（最高分）会保存在浏览器的本地存储中
- 停止服务器：在命令行窗口按 `Ctrl+C`
- 如果端口8000被占用，可以修改启动脚本中的端口号
